Tile Coding¶

Tile coding is an innovative way of discretizing a continuous space that enables better generalization compared to a single grid-based approach. The fundamental idea is to create several overlapping grids or tilings; then for any given sample value, you need only check which tiles it lies in. You can then encode the original continuous value by a vector of integer indices or bits that identifies each activated tile.

3. Tiling¶

Let's first design a way to create a single tiling for a given state space. This is very similar to a uniform grid! The only difference is that you should include an offset for each dimension that shifts the split points.

For instance, if low = [-1.0, -5.0], high = [1.0, 5.0], bins = (10, 10), and offsets = (-0.1, 0.5), then return a list of 2 NumPy arrays (2 dimensions) each containing the following split points (9 split points per dimension):

[array([-0.9, -0.7, -0.5, -0.3, -0.1,  0.1,  0.3,  0.5,  0.7]),
 array([-3.5, -2.5, -1.5, -0.5,  0.5,  1.5,  2.5,  3.5,  4.5])]

Notice how the split points for the first dimension are offset by -0.1, and for the second dimension are offset by +0.5. This might mean that some of our tiles, especially along the perimeter, are partially outside the valid state space, but that is unavoidable and harmless.

def create_tiling_grid(low, high, bins=(10, 10), offsets=(0.0, 0.0)):
    """Define a uniformly-spaced grid that can be used for tile-coding a space.
    
    Parameters
    ----------
    low : array_like
        Lower bounds for each dimension of the continuous space.
    high : array_like
        Upper bounds for each dimension of the continuous space.
    bins : tuple
        Number of bins or tiles along each corresponding dimension.
    offsets : tuple
        Split points for each dimension should be offset by these values.
    
    Returns
    -------
    grid : list of array_like
        A list of arrays containing split points for each dimension.
    """
    arrays = []
    for dim in range(len(bins)):
        array = np.linspace(low[dim], high[dim], bins[dim] + 1)[1:-1] + offsets[dim]
        arrays.append(array)
    return arrays

def create_tilings(low, high, tiling_specs):
    """Define multiple tilings using the provided specifications.

    Parameters
    ----------
    low : array_like
        Lower bounds for each dimension of the continuous space.
    high : array_like
        Upper bounds for each dimension of the continuous space.
    tiling_specs : list of tuples
        A sequence of (bins, offsets) to be passed to create_tiling_grid().

    Returns
    -------
    tilings : list
        A list of tilings (grids), each produced by create_tiling_grid().
    """
    tilings = []
    for bins, offsets in tiling_specs:
        tilings.append(create_tiling_grid(low, high, bins, offsets))
    return tilings

4. Tile Encoding¶

Implement the following to produce a vector that contains the indices for each tile that the input state value belongs to. The shape of the vector can be the same as the arrangment of tiles you have, or it can be ultimately flattened for convenience.

You can use the same discretize() function here from grid-based discretization, and simply call it for each tiling.

In [51]:
def discretize(sample, grid):
    """Discretize a sample as per given grid.
    
    Parameters
    ----------
    sample : array_like
        A single sample from the (original) continuous space.
    grid : list of array_like
        A list of arrays containing split points for each dimension.
    
    Returns
    -------
    discretized_sample : array_like
        A sequence of integers with the same number of dimensions as sample.
    """
    binned = []
    for dim_s, dim_g in zip(sample, grid):
        binned.append(int(np.digitize(dim_s, dim_g)))
    return tuple(binned)


def tile_encode(sample, tilings, flatten=False):
    """Encode given sample using tile-coding.
    
    Parameters
    ----------
    sample : array_like
        A single sample from the (original) continuous space.
    tilings : list
        A list of tilings (grids), each produced by create_tiling_grid().
    flatten : bool
        If true, flatten the resulting binary arrays into a single long vector.

    Returns
    -------
    encoded_sample : list or array_like
        A list of binary vectors, one for each tiling, or flattened into one.
    """
    encoded_sample = []
    for grid in tilings:
        encoded_sample.append(discretize(sample, grid))
    if flatten:
        return np.concatenate(encoded_sample)
    else:
        return encoded_sample

    # Show tiling grids
    ax = visualize_tilings(tilings)
    
    # If bounds (low, high) are specified, use them to set axis limits
    if low is not None and high is not None:
        ax.set_xlim(low[0], high[0])
        ax.set_ylim(low[1], high[1])
    else:
        # Pre-render (invisible) samples to automatically set reasonable axis limits, and use them as (low, high)
        ax.plot(samples[:, 0], samples[:, 1], 'o', alpha=0.0)
        low = [ax.get_xlim()[0], ax.get_ylim()[0]]
        high = [ax.get_xlim()[1], ax.get_ylim()[1]]

    tilings_extended = [np.hstack((np.array([low]).T, grid, np.array([high]).T)) for grid in tilings]  # add low and high ends
    tile_centers = [(grid_extended[:, 1:] + grid_extended[:, :-1]) / 2 for grid_extended in tilings_extended]  # compute center of each tile
    tile_toplefts = [grid_extended[:, :-1] for grid_extended in tilings_extended]  # compute topleft of each tile
    tile_bottomrights = [grid_extended[:, 1:] for grid_extended in tilings_extended]  # compute bottomright of each tile

    prop_cycle = plt.rcParams['axes.prop_cycle']
    colors = prop_cycle.by_key()['color']
    for sample, encoded_sample in zip(samples, encoded_samples):
        for i, tile in enumerate(encoded_sample):
            # Shade the entire tile with a rectangle
            topleft = tile_toplefts[i][0][tile[0]], tile_toplefts[i][1][tile[1]]
            bottomright = tile_bottomrights[i][0][tile[0]], tile_bottomrights[i][1][tile[1]]
            ax.add_patch(Rectangle(topleft, bottomright[0] - topleft[0], bottomright[1] - topleft[1],
                                   color=colors[i], alpha=0.33))

            # In case sample is outside tile bounds, it may not have been highlighted properly
            if any(sample < topleft) or any(sample > bottomright):

                ax.plot(cx, cy, 's', color=colors[i])
    
    # Finally, plot original samples
    ax.plot(samples[:, 0], samples[:, 1], 'o', color='r')

    ax.margins(x=0, y=0)  # remove unnecessary margins
    ax.set_title("Tile-encoded samples")

5. Q-Table with Tile Coding¶

The next step is to design a special Q-table that is able to utilize this tile coding scheme. It should have the same kind of interface as a regular table, i.e. given a <state, action> pair, it should return a <value>. Similarly, it should also allow you to update the <value> for a given <state, action> pair (note that this should update all the tiles that <state> belongs to).

The <state> supplied here is assumed to be from the original continuous state space, and <action> is discrete (and integer index). The Q-table should internally convert the <state> to its tile-coded representation when required.

class QTable:
    """Simple Q-table."""

    def __init__(self, state_size, action_size):
        """Initialize Q-table.
        
        Parameters
        ----------
        state_size : tuple
            Number of discrete values along each dimension of state space.
        action_size : int
            Number of discrete actions in action space.
        """
        self.state_size = state_size
        self.action_size = action_size

        
        # Note: If state_size = (9, 9), action_size = 2, q_table.shape should be (9, 9, 2)
        # initialize action value table
        self.q_table = np.zeros(shape=(self.state_size + (self.action_size,)))
        print("QTable(): size =", self.q_table.shape)


class TiledQTable:
    """Composite Q-table with an internal tile coding scheme."""
    
    def __init__(self, low, high, tiling_specs, action_size):
        """Create tilings and initialize internal Q-table(s).
        
        Parameters
        ----------
        low : array_like
            Lower bounds for each dimension of state space.
        high : array_like
            Upper bounds for each dimension of state space.
        tiling_specs : list of tuples
            A sequence of (bins, offsets) to be passed to create_tilings() along with low, high.
        action_size : int
            Number of discrete actions in action space.
        """
        self.tilings = create_tilings(low, high, tiling_specs)
        self.state_sizes = [tuple(len(splits)+1 for splits in tiling_grid) for tiling_grid in self.tilings]
        self.action_size = action_size
        self.q_tables = [QTable(state_size, self.action_size) for state_size in self.state_sizes]
        print("TiledQTable(): no. of internal tables = ", len(self.q_tables))
    
    def get(self, state, action):
        """Get Q-value for given <state, action> pair.
        
        Parameters
        ----------
        state : array_like
            Vector representing the state in the original continuous space.
        action : int
            Index of desired action.
        
        Returns
        -------
        value : float
            Q-value of given <state, action> pair, averaged from all internal Q-tables.
        """
        # Encode state to get tile indices
        state_encodings = tile_encode(state, self.tilings)
        # Retrieve q-value for each tiling, and return their average
        q_value = 0.0
        # add all corresponding state action values for each tiling
        for encoded_state, q_table in zip(state_encodings, self.q_tables):
            q_value += q_table.q_table[encoded_state + (action,)]
        # get average of all action values
        q_value /= len(state_encodings)
        return q_value

    def update(self, state, action, q_value, alpha=0.1):
        """Soft-update Q-value for given <state, action> pair to value.
        
        Instead of overwriting Q(state, action) with value, perform soft-update:
            Q(state, action) = alpha * value + (1.0 - alpha) * Q(state, action)
        
        Parameters
        ----------
        state : array_like
            Vector representing the state in the original continuous space.
        action : int
            Index of desired action.
        value : float
            Desired Q-value for <state, action> pair.
        alpha : float
            Update factor to perform soft-update, in [0.0, 1.0] range.
        """
        # Encode state to get tile indices
        state_encodings = tile_encode(state, self.tilings)
        # Update q-value for each tiling by update factor alpha
        for encoded_state, q_table in zip(state_encodings, self.q_tables):
            # current action value given tiling
            curr_q = q_table.q_table[encoded_state + (action, )]
            # update encoded state's action value 
            q_table.q_table[encoded_state + (action, )] = alpha * q_value + (1.0 - alpha) * curr_q

class QLearningAgent:
    """Class with environment interaction methods"""
    
    # define all needed variables for Q-Learning
    def __init__(self, env, tiled_q, alpha=0.01, gamma=1.0, eps=1.0, eps_decay=0.9995, min_eps=0.01, seed=714):
        self.env = env # environment
        self.tiled_q = tiled_q # tiled Q tables for all tiling_specs
        self.state_sizes = tiled_q.state_sizes # number of bins used to discretize
        self.action_size = self.env.action_space.n # number of actions possible
        self.seed = np.random.seed(seed) # set random seed for epsilon greedy action selection
        self.alpha = alpha # step size for each update
        self.gamma = gamma # discount rate 
        self.eps = eps # initiate decaying epsilon
        self.new_eps = eps # set constant epsilon for new training session
        self.eps_decay = eps_decay # constant for exploration decay and higher exploitation 
        self.min_eps = min_eps # guarantees a minimum exploration possibility
    
    def new_episode(self, first_state):
        """Update Epsilon Decay and Return First Action of Episode"""
        # decay epsilon after every episode in training session
        self.eps *= self.eps_decay
        # if epsilon decays past epsilon minimum, select epsilon minimum
        self.eps = max(self.eps, self.min_eps)
        # keep track of first state for action values update
        self.state = first_state
        # check all action values for state
        Q_state = [self.tiled_q.get(first_state, action) for action in range(self.action_size)]
        # select first action as greedy action
        self.action = np.argmax(Q_state)
        # return first action of episode
        return self.action
    
    def reset_eps(self, epsilon=None):
        """Reset Epsilon to New Epsilon or User Defined""" 
        # if user did not define new epsilon, use default
        if epsilon is None:
            self.eps = self.new_eps
        # user defined epsilon
        else:
            self.eps = epsilon
            
    def update_q(self, next_state, reward=None):
        """Return Next Action and Update Previous Action Value"""   
        # next state action values
        Q_state = [self.tiled_q.get(next_state, action) for action in range(self.action_size)]
        # set next state greedy action 
        greedy_action = np.argmax(Q_state)
        # initialize action space probabilities
        eps_probs = np.ones(self.action_size) * self.eps / self.action_size
        # update greedy action probability to be more probable even when exploring
        eps_probs[greedy_action] = 1 - self.eps + self.eps / self.action_size
        
        # get action value for last action using reward and next state greedy action value
        update_q = reward + self.gamma * max(Q_state)
        # update tiled_q for corresponding state and action
        self.tiled_q.update(self.state, self.action, update_q, self.alpha)
        
        # select next action using epsilon greedy probabilities
        next_action = np.random.choice(np.arange(self.action_size), p=eps_probs)
        
        # update action and state
        self.state = next_state
        self.action = next_action
        
        # return next action
        return next_action

def run(agent, env, num_episodes=10000):
    """Run agent & return episode scores"""
    
    #initialize episode scores
    scores = []
    # initialize max average score obtained by agent
    max_avg_score = -np.inf
    # begin training
    for i_episode in range(1, num_episodes+1):
        # initialize episode
        state = env.reset()
        # get episode's first action
        next_action =  agent.new_episode(state)
        # intialize episode's total reward
        total_reward = 0
        # initialize terminal state to False
        terminal_s = False

        # continue episode if not in a terminal state
        while not terminal_s:
            # take action and record outcomes
            next_state, reward, terminal_s, other = env.step(next_action)
            # update episode's total reward w/ action's reward
            total_reward += reward
            # update action values and get next action
            next_action = agent.update_q(next_state, reward)

        # save episode's total reward to scores
        scores.append(total_reward)

        # print episode stats
        if len(scores) > 100:
            # get averageget of last 100 episode scores
            avg_score = np.mean(scores[-100:])
            # keep track of highest average score
            if avg_score > max_avg_score:
                max_avg_score = avg_score
        if i_episode % 100 == 0:
            print("\rEpisode {}/{} | Max Average Score: {}".format(i_episode, num_episodes, max_avg_score), end="")
            sys.stdout.flush()
    return scores
